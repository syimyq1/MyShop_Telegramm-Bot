# Telegram Store Bot 🛒

## Description

A Telegram store bot built with Python that allows users to browse products and interact with an online shop directly through Telegram.

The bot provides a simple way to manage products and communicate with customers using Telegram commands.

## Features

- 🛍 View available products
- 📦 Product information and prices
- 🛒 Add products to cart
- 💬 User interaction through Telegram commands
- ⚙️ Easy product management
- 💾 Data storage support

## Technologies

- Python
- Telegram Bot API
- pyTelegramBotAPI
- SQLite (if used)
